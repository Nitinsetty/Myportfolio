Task List - Agendador de Tarefas

Visão Geral

Porjeto simples e fácil com HTML, CSS E JavaScript, criei uma div com um botão e um input para add as tarefas, uma div que guardaria as tarefas adicionadas e uma div principal que uniu as duas anteriores. Usei CSS para estilizar a pagina e alinhar todos os elementos ao centro para uma melhor vizualização. Com JavaScript criei funções para adição, validação e inserção, e deleção das tarefas
